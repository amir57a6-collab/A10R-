Frequently Asked Questions

How does the image upload feature work?+

We use a multi-modal AI model (Gemini) that can understand both text and images. When you upload an image, it's sent along with your text prompt to the AI, which uses it as a powerful visual guide to generate more accurate code.

Is my generated code saved securely?+

Yes. When you are signed in, your code history is saved to your personal account using Google's secure Firebase platform. Only you can access your creation history.

Can I use the generated code in commercial projects?+

Absolutely. You own the code you generate with A10r. Feel free to use it in any personal or commercial projects without attribution.
